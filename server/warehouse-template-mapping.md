# Warehouse & Distribution Template: Threat → Control → Question Mapping

## Purpose
Map warehouse-specific threats to controls and define questions assessing effectiveness.

---

## 1. Cargo Theft
**Threat:** Theft of goods during transport, loading/unloading, or from yard

**Mitigating Controls:**
- Perimeter Fencing - Chain Link
- Vehicle Gates - Automated
- CCTV System - IP/Network
- Loading Dock Security Seals
- Vehicle Inspection Procedures

**Survey Questions:**
1. **Perimeter Security**
   - Q: "Is the warehouse perimeter protected by chain-link fencing or similar barrier?"
   - Type: Yes/No
   - If Yes: "Rate fence condition and effectiveness (1-5)"
   - Evidence: Photo of perimeter fence
   - Control: Perimeter Fencing - Chain Link

2. **Vehicle Gate Control**
   - Q: "Are vehicle gates automated and access-controlled for entry/exit?"
   - Type: Yes/No
   - If Yes: "Rate effectiveness of gate access control (1-5)"
   - Evidence: Photo of vehicle gate system
   - Control: Vehicle Gates - Automated

3. **Yard & Dock Surveillance**
   - Q: "Is CCTV surveillance installed covering loading docks, yard, and trailer parking areas?"
   - Type: Yes/No
   - If Yes: "Rate CCTV coverage and quality (1-5)"
   - Evidence: Camera coverage map
   - Control: CCTV System - IP/Network

4. **Shipment Seals**
   - Q: "Are tamper-evident seals used on all trailers and containers?"
   - Type: Yes/No
   - If Yes: "Rate seal inspection and logging procedures (1-5)"
   - Evidence: Seal log sample
   - Control: Loading Dock Security Seals

5. **Vehicle Inspection**
   - Q: "Are incoming and outgoing vehicles inspected for unauthorized cargo?"
   - Type: Yes/No
   - If Yes: "Rate thoroughness of vehicle inspection procedures (1-5)"
   - Evidence: Inspection checklist
   - Control: Vehicle Inspection Procedures
