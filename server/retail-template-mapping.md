# Retail Store Template: Threat → Control → Question Mapping

## Purpose
Map each threat to its mitigating controls and define survey questions that assess control effectiveness for risk calculations.

---

## 1. Organized Retail Crime
**Threat:** Professional theft rings targeting merchandise for resale

**Mitigating Controls:**
- Electronic Article Surveillance (EAS)
- CCTV System - IP/Network
- Security Officers - Unarmed
- Security Awareness Training

**Survey Questions:**
1. **EAS System Assessment**
   - Q: "Does the store have an Electronic Article Surveillance (EAS) system installed at exits?"
   - Type: Yes/No
   - If Yes: "Rate the effectiveness of the EAS system (1=Ineffective, 5=Highly Effective)"
   - Evidence: Photo of EAS gates
   - Control: Electronic Article Surveillance (EAS)

2. **CCTV Coverage**
   - Q: "Is CCTV surveillance installed covering all customer areas, exits, and high-value merchandise?"
   - Type: Yes/No
   - If Yes: "Rate the CCTV system's effectiveness in deterring theft (1-5)"
   - Evidence: Photo of camera coverage map
   - Control: CCTV System - IP/Network

3. **Loss Prevention Staffing**
   - Q: "Are uniformed security officers or loss prevention personnel present during operating hours?"
   - Type: Yes/No
   - If Yes: "Rate their effectiveness in preventing organized retail crime (1-5)"
   - Evidence: Photo of security presence
   - Control: Security Officers - Unarmed
